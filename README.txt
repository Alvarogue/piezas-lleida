
INSTRUCCIONES PARA PUBLICAR TU WEB EN GITHUB PAGES:

1. Inicia sesión en https://github.com/
2. Haz clic en "+" (arriba a la derecha) y selecciona "New repository"
   - Nombre del repositorio: piezas-lleida (o el que prefieras)
   - Ponlo como público
   - No marques ninguna opción adicional
3. Una vez creado el repositorio, haz clic en "Add file" > "Upload files"
4. Arrastra todos los archivos de esta carpeta (.zip descomprimido) y haz clic en "Commit changes"
5. Luego ve a "Settings" > "Pages" (barra lateral izquierda)
   - En "Source", selecciona "Deploy from a branch" y elige "main" y la carpeta raíz (/)
   - Guarda los cambios
6. Espera unos segundos, tu web estará disponible en una URL como:
   https://TU_USUARIO.github.io/piezas-lleida

7. ¡Listo! Ya puedes compartir tu web y agregarla a Google Search Console para que aparezca en Google.
